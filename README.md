# shuangxiang
3月19日的一条龙升级，内含隐藏配置。可以利用【待处理】目录内的账号，刷telegram ads广告（又名sponsor message）。

可以用这个配置文件来做（我这个配置是使用双向号来跑）。
关键配置：

# sponsor feature
# 是否开启 sponsor 功能
is_check_sponsor = true
# 检查sponsor的账号比例
view_sponsor_rate = 100
# sponsor的target机器人列表
sponsor_target_file = "./【sponsor】的目标在这里.txt"
# 点击sponsor的账号比例
click_sponsor_rate = 4
